# 个人网站

基于 Jekyll 的个人网站，使用 no-style-please 主题。

## 功能特点

- 快速、简洁的设计
- 支持数学公式渲染
- 响应式布局
- 支持暗色/亮色主题自动切换
- SEO 优化
- RSS 订阅支持

## 本地开发

1. 安装 Ruby 和 Jekyll
2. 克隆仓库
3. 运行 `bundle install`
4. 运行 `bundle exec jekyll serve`
5. 访问 http://localhost:4000

## 自定义

- 修改 `_config.yml` 更新网站信息
- 编辑 `index.md` 更新主页内容
- 编辑 `about.md` 更新关于页面
- 修改 `_data/menu.yml` 自定义导航菜单
- 在 `_posts` 目录下添加博客文章
